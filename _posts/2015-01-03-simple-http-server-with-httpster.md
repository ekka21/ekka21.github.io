Simple http server for quick loading of content 

<pre>npm install -g httpster
</pre>

Using it?

From any directory where you want to have an http service running, just run

<pre>httpster
</pre>

or

<pre>httpster -p 8080 -d /home/somedir/public_html
</pre>
